# 错误记录

程序运行、生成安装器、WinRAR 封装或定时切换失败时，会把错误追加到这里。

## 2026-04-13 修复记录

```text
问题：
1. 鼠标文件默认存放在 LocalAppData，存在误删和迁移不清晰的问题。
2. 新增资源包不应直接变成 03/04 默认方案，需要作为资源库方案按需添加。
```

## 2026-04-14 卸载、自启动降噪与发布包命名

```text
问题：
1. 卸载程序点击后可能看不到窗口，且安装目录清理脚本会在卸载程序仍占用文件时提前删除，导致清理失败。
2. 当前系统创建 schtasks 登录任务返回“拒绝访问”，每次应用或关闭时都会重复写入错误记录。
3. 重启后后台可能因为旧 background.pid 里的 PID 被系统复用而直接退出。
4. Release 资产名称需要固定为“鼠标指针配置生成器_安装程序.exe”和“鼠标指针配置生成器_绿色程序.exe”，并且必须同时上传。

处理：
1. 卸载窗口居中置顶显示，卸载前终止后台进程；安装目录清理改用隐藏 PowerShell 循环重试，避开中文 CMD 路径和文件占用。
2. 任务计划拒绝访问时只记录一次，并标记为禁用尝试；后续使用 HKCU Run 项和启动文件夹快捷方式完成自启动。
3. background.pid 改为记录 JSON，校验 PID 对应的可执行文件路径，避免 PID 复用误判。
4. Release workflow 改用 gh release create 显式上传两份指定名称的 EXE。
```
