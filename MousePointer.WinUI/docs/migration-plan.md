1. 导入 `.cur`、`.ani`、图片、`.zip`、`.rar`、`.7z`、`.exe`、文件夹。
2. 检查 `Arrow`、`NWPen`、`Hand`、`Crosshair`、`Pin`、`Person` 是否落位正确。
3. 保存并应用部分方案，确认未配置角色不会被覆盖。
4. 调整鼠标大小并应用，确认系统设置同步。
5. 保存计时切换，分别运行 `--background` 和 `--tray`。
6. 生成安装包，在另一台 Windows 环境运行。
7. 验证恢复上一份鼠标方案。
