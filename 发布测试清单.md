# 发布测试清单

用于每次打包发布前检查。能自动检查的由开发者执行；需要真实系统环境的项目由使用者手动确认。

## 基础启动

- [ ] Win10 普通用户权限启动。
- [ ] Win11 普通用户权限启动。
- [ ] 程序窗口首次打开居中显示。
- [ ] 第二次双击程序时唤起已有窗口，不再打开新的完整界面。
- [ ] 首次启动会询问是否添加桌面快捷方式。
- [ ] 选择“不添加”后，设置页仍可点击“添加桌面快捷方式”。

## 方案编辑

- [ ] 新建方案后可导入单个 `.cur` 文件。
- [ ] 新建方案后可导入单个 `.ani` 文件，并能实时预览动画。
- [ ] 拖入图片能替换当前鼠标状态。
- [ ] 方案切换后，下方文件配置同步变化。
- [ ] 修改任意指针状态后无需点击保存，重启程序后仍能看到修改。
- [ ] 删除方案后切换到上一个方案，而不是固定跳到第一个。

## 资源导入

- [ ] 导入 `.zip` 鼠标包。
- [ ] 导入 `.rar` 鼠标包。
- [ ] 导入 `.7z` 鼠标包。
- [ ] 导入自解压或 PyInstaller `.exe` 鼠标包。
- [ ] 导入文件夹。
- [ ] 一个压缩包内有多个 `.inf` 时可批量导入。
- [ ] 重复方案会提示并可选择跳过或导入副本。
- [ ] 导入完成后显示成功、跳过、失败数量。
- [ ] 导入失败时错误写入 `错误记录.txt`。

## 应用和安装包

- [ ] 方案完整时可正常应用。
- [ ] 方案缺少部分鼠标状态时，应用前会提示缺失状态。
- [ ] 鼠标大小滑条选择后，应用方案会按当前大小写入系统。
- [ ] 应用期间显示等待状态，不像卡死。
- [ ] 生成安装包时可选择保存位置。
- [ ] 创建成功后才打开目标文件夹。
- [ ] 生成的安装器不显示 Python 黑色窗口。
- [ ] 生成的安装器图标来自“正常选择”指针。
- [ ] 安装器运行后可应用对应方案和大小。

## 截图导出

- [ ] 无鼠标指针资源时不导出空截图。
- [ ] 有静态指针时导出 `.gif`。
- [ ] 有动态 `.ani` 指针时导出的 `.gif` 能播放动画。
- [ ] 每行 5 个图标排列。

## 后台和自启动

- [ ] 勾选自启动后台并保存后，设置页显示自启动状态正常。
- [ ] 点击“修复自启动”后能重新写入启动项。
- [ ] 关闭窗口后保留托盘图标。
- [ ] 托盘左键可以打开主窗口。
- [ ] 托盘右键“打开”可以打开主窗口。
- [ ] 托盘右键“退出”会退出后台。
- [ ] 开启“隐藏任务栏”后关闭窗口不显示托盘，只保留后台。
- [ ] 关闭“隐藏任务栏”后托盘图标能恢复显示。
- [ ] 重启 Windows 后不弹主窗口，只显示托盘或后台状态。

## 方案切换

- [ ] 时间切换开启后，到点立即应用。
- [ ] 星期切换开启后，当天配置立即应用。
- [ ] 计时切换开启后按间隔切换。
- [ ] 中英文输入状态切换延迟可接受，不影响正常输入。
- [ ] 三种切换模式只能启用一种。

## 更新和发布

- [ ] 设置页显示版本号和当前提交。
- [ ] 检查更新无 Release 时提示清晰，不闪退。
- [ ] 有 Release 时能下载对应 exe。
- [ ] 自动更新不会弹 Python 黑色窗口。
- [ ] `dist` 和 `release-assets` 中的 exe 哈希一致。

## 需要使用者重点手动测试

- Win10/Win11 真实机器。
- 重启后的自启动状态。
- 不同 DPI 缩放：100%、125%、150%。
- 常见杀毒软件或 Windows 安全中心是否拦截启动项。
- 不同来源鼠标包的导入兼容性。
