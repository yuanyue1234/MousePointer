# 鼠标指针配置管理器产品规格

## 产品目标

让普通用户可以导入、预览、应用和管理 Windows 鼠标指针方案；让鼠标指针作者可以整理方案、替换单个角色、生成可分发安装包。

当前成品主线继续采用 Python 框架，不引入 React、WebView2 或 C# 作为正式产品入口。

## 核心用户路径

- 导入资源包：支持文件夹、`.zip`、`.rar`、`.7z`、安装器和单个指针文件。
- 编辑方案：按 Windows 鼠标角色替换 `.cur`、`.ani` 或图片素材，预览失败不影响应用。
- 应用方案：手动应用、资源库应用、自动切换应用都必须更新当前方案状态。
- 管理资源库：列表和宫格视图都支持选择、单个应用、批量删除。
- 发布检查：生成绿色版或安装器前，必须完成基础导入、应用、恢复、后台和自启动检查。

## 体验原则

- 预览图优先展示素材本体，辅助标识不能遮挡图标。
- 高频操作放在固定位置，资源库只保留一个全局应用和一个全局删除按钮。
- 复杂操作给清楚反馈，失败提示要能帮助定位，不用弹窗打断普通浏览。
- 大文件拆分要在测试保护下进行，避免为重构牺牲成品稳定性。

## 非目标

- 本阶段不重写为 React/WebView2/C#。
- 本阶段不做大型视觉换皮。
- 不把 WinUI 目录作为正式产品入口继续推进。
