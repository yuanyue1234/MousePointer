# 资源库交互规格

## 页面结构

- 顶部保留在线资源库、导入资源、导入文件夹、刷新、恢复上一份鼠标方案、列表/宫格切换。
- 中间展示资源方案。列表适合快速扫描，宫格适合视觉浏览。
- 底部右侧只保留一个 `应用` 和一个 `删除` 操作按钮。

## 选择规则

- 点击卡片或复选框切换选中状态。
- 列表视图和宫格视图共享同一组选中状态。
- 刷新后自动移除已经不存在的选中项。
- 选中 0 个方案：`应用` 和 `删除` 不可用。
- 选中 1 个方案：`应用` 和 `删除` 可用。
- 选中多个方案：只允许 `删除`，`应用` 不可用。

## 应用规则

- 应用只接受单选方案。
- 应用成功后必须让方案页下拉框、当前方案状态和系统鼠标保持一致。
- 应用失败时不能提前写入当前方案状态。

## 删除规则

- 删除支持多选批量操作。
- 删除不直接永久移除，而是移动到 `build/resource_trash`。
- 删除后清空对应选中状态，并刷新资源库和方案列表。
- 删除完成后提示回收区位置，方便误删恢复。

## 动/静标识

- 方案编辑页：在角色名称旁显示 `动` 或 `静` 小标识。
- 替换资源池：在图标下方显示 `动` 或 `静` 小标识。
- 资源库：预览小图不显示逐个标识，只在方案名旁显示汇总，例如 `动 3`、`静 14`。
- 标识不得覆盖鼠标图标、删除按钮或主要文字。
