# 鼠标指针配置管理器

Windows 鼠标指针方案图形化工具。宗旨是让新手小白也能用，同时让鼠标指针制作者方便编辑、预览、应用和生成安装器。

## 当前版本

版本：`2.0.0`

## 主要功能

- 鼠标方案页支持新建、重命名、删除、导入、自动保存、应用、生成安装包和截图导出。
- 方案配置支持 `.cur`、`.ani`、常见图片、`.zip`、`.rar`、`.7z`、`.exe` 和文件夹导入。
- 导入资源包时会识别 `.inf`，支持一个压缩包内包含多份鼠标方案，并在完成后汇总成功、跳过和失败数量。
- 方案不完整时，应用或生成安装包前会提示缺少哪些鼠标状态，避免用户误以为程序失效。
- 右侧实时预览支持静态和动态鼠标指针，动态 `.ani` 会播放。
- 鼠标大小滑条用于预判实际尺寸；应用方案和生成安装包会携带当前大小，并先刷新系统鼠标参数。
- 截图导出保存为 `.gif`，有动态指针时会导出多帧动图。
- 设置页可修改鼠标文件存放位置，默认位于 `%APPDATA%\MouseCursorThemeBuilder\mouse_files`。
- 设置页提供添加桌面快捷方式、检测更新、自启动后台、隐藏任务栏和自启动修复。
- 首次手动打开会询问是否添加桌面快捷方式，后续也可在设置中重新添加。
- 开启自启动后台后，关闭窗口会保留后台或托盘；托盘左键打开窗口，右键菜单提供打开、隐藏任务栏和退出。
- 启动时默认居中显示；重复打开程序会尝试唤起已有窗口，减少重复实例。
- 错误会写入 `错误记录.txt`，日志超过 2MB 后自动归档，最多保留 5 份且保留 30 天。

## 运行源码

```powershell
.\.venv\Scripts\python.exe -m pip install -r requirements.txt
.\.venv\Scripts\python.exe main.py
```

## 打包

当前标准版使用 PyInstaller onefile：

```powershell
.\.venv\Scripts\python.exe -m PyInstaller -y --clean "鼠标指针配置生成器_绿色程序.spec"
```

输出位置：

```text
dist\鼠标指针配置生成器_绿色程序.exe
release-assets\鼠标指针配置生成器_绿色程序.exe
```
